Incremental-Tutorial
Aperçu final du jeu que l'on va créer

cookies.
Explication

Voici le code Javascript du jeu :

var cookies = 0;

function gagnerCookie() {
	cookies += 1;
	document.getElementById("affichageCookies").innerHTML = cookies;
};

window.onload = function() {
	document.getElementById("affichageCookies").innerHTML = cookies;
};

Voici le code HTML du jeu :

<button onclick="gagnerCookie()">+1 cookie</button><br>
<span id="affichageCookies"></span> cookies.

En Javascript, on commence toujours par créer des variables. Ces variables permettent de stocker des données comme des chiffres, des lettres ou même des phrases. Afin de différencier les variables, on doit toujours leur attribuer un nom. Ce nom peut être composé de lettre et de chiffre uniquement. Il est préférable de donner une valeur de base à notre variable après l'avoir créée comme par exemple, un 0. Ce qui voudra dire que notre variable sera égale à 0. Une fois cette variable créée, on termine par un point-virgule afin de dire au Javascript que la création de notre variable s'arrête ici.

Dans notre cas, nous avons besoin de stocker le nombre de cookies, alors nous allons créer une variable appellée cookies comme ceci : 

var cookies = 0;

Ensuite nous allons utiliser les fonctions. Elles sont utilisées pour demander à l'ordinateur d'effectuer les lignes de commandes qui se trouvent dans ces fonctions. Pour créer une fonction, on commence par dire au Javascript que nous voulons déclarer une fonction en commençant par le mot function suivi du nom qu'on souhaite lui donner, comme pour les variables. Ici, nous allons l'appeller gagnerCookie, puis on termine par mettre des parenthèses (nous verrons plus tard a quoi servent-elles). Ensuite, nous devons mettre des accolades qui vont désigner les limites de la fonction, les lignes de code a éxecuter dans cette fonction. Ce qui donne : 

function gagnerCookie() {
};

Maintenant nous devons dire ce qu'il va se passer lorsque nous demandons à l'ordinateur d'éxécuter la fonction gagnerCookie, pour cela nous allons créér de nouvelles lignes de code. Notre but, est d'augmenter notre nombre de cookies lorsque nous appellons cette fonction, par exemple lorsqu'on clique sur le bouton. Alors nous allons devoir modifier la variable cookies lorsqu'on éxécute la fonction gagnerCookie. Pour cela, rien de plus simple, on ajoute dans notre fonction gagnerCookie la ligne de code suivante entre les accolades : cookies = cookies + 1;

Ce qui nous donne maintenant ceci : 

function gagnerCookie() {
	cookies += 1;
};

La ligne de code cookies = cookies + 1; demande a l'ordinateur de : prendre la variable cookies, puis de lui dire d'ajouter + 1 au nombre de cookies, rien de plus simple.

La prochaine étape est d'ajouter l'affichage du nombre de cookies sur la page. Pour cela, on créé un élément avec le tag <span></span> et on lui donne un nom ou alors un id, dans l'attribut id. Par exemple, nous avons notre tag span et allons lui donner le nom affichageCookies, ce qui nous donne : 

<span id="affichageCookies"></span>

Notre tag span est maintenant identifiable grâce à son nom affichageCookies, donc nous allons pouvoir l'utiliser dans le Javascript. Si on veut actualiser notre nombre de cookies qui sera inclu dans le tag span, il va falloir ajouter une ligne de code en Javascript dans notre fonction gagnerCookie, cette ligne de code : 

document.getElementById("affichageCookies").innerHTML = cookies;

Que fait concrètement cette ligne de code ?

Cette ligne de code va chercher dans le document HTML (document.) un élément, ou un tag si vous préférez qui porte un id (getElementById) que vous précisez entre les parenthèses (("affichageCookies")), puis ensuite on ajoute innerHTML. On fini par mettre le symbole = qui va demander au Javascript d'afficher ce qu'il y a entre l'élément/le tag qui porte pour id affichageCookies. Nous ce qu'on veut c'est juste afficher le nombre de cookies, et ce nombre de cookies est stocké dans notre variable appellée cookies. Alors nous allons tout simplement mettre cookies après le symbole =. Ce qui devrait donner : 

function gagnerCookie() {
	cookies += 1;
	document.getElementById("affichageCookies").innerHTML = cookies;
};

Presque la fin, il nous reste plus qu'a ajouter le bouton dans notre HTML. Pour cela, on peut créer un simple bouton avec le tag <button></button>. Ensuite on va ajouter un attribut à ce bouton, l'attribut onclick. Cet attribut permet de demander au bouton que, lorsqu'on clique dessus, il exécute la fonction définie. Pour cela, on met seulement le nom de la fonction dans notre attribut onclick suivi d'un égal, puis on met entre les apostrophes le nom de la variable sans oublier les parenthèses ! On peut par exemple dire pour notre jeu : 

<button onclick="gagnerCookie()"></button>

On peut mettre du contenu dans le bouton, du texte qui s'affichera dans le bouton comme ceci : 

<button onclick="gagnerCookie()">+1 cookie</button>

Notre jeu est prêt, la base est maintenant créée, dans le prochain tutoriel nous allons voir comme créer un maagasin afin d'acheter des builds qui nous permetteront de gagner des cookies toutes les secondes.
