cookies
Explication

Voici le code du jeu :

var cookies = 0;

function gagnerCookie() {
	cookies += 1;
	document.getElementById("affichageCookies").innerHTML = cookies;
};

window.onload = function() {
	document.getElementById("affichageCookies").innerHTML = cookies;
};

En Javascript, on commence toujours par créer des variables. Ces variables permettent de stocker des données comme des chiffres, des lettres ou même des phrases. Afin de différencier les variables, on doit toujours leur attribuer un nom. Ce peut être composé de lettre et de chiffre uniquement. Il est préférable de donner une valeur de base à notre variable après l'avoir créer comme par exemple, un 0. Une fois cette variable créée, on termine par un point-virgule afin de dire au Javascript que la création de notre variable s'arrête ici.

Dans notre cas, nous avons besoin de stocker le nombre de cookies, alors nous allons créer une variable appellée cookies comme ceci : 

var cookies = 0;

Ensuite nous allons utiliser les fonctions. Elles sont utilisées pour demander à l'ordinateur d'effectuer les lignes de commandes qui se trouvent dans ces fonctions. Pour créer une fonction, on commence par dire au Javascript que nous voulons déclarer une fonction en commençant par le mot function suivi du nom qu'on souhaite lui donner, comme pour les variables. Ici, nous allons l'appeller gagnerCookie, puis on termine par mettre des parenthèses (nous verrons plus tard a quoi servent-elles). Ensuite, nous devons mettre des accolades qui vont désigner les limites de la fonction, les lignes de code a éxecuter dans cette fonction. Ce qui donne : 

function gagnerCookie() {

};

Maintenant nous devons dire ce qu'il va se passer lorsque nous demandons à l'ordinateur d'éxécuter la fonction gagnerCookie, pour cela nous allons créer de nouvelles lignes de code. Notre but, est d'augmenter notre nombre de cookies lorsque nous appellons cette fonction, alors nous allons devoir modifier la variable cookies lorsqu'on éxécute la fonction gagnerCookie. Pour cela, rien de plus simple, on ajoute dans notre fonction gagnerCookie la ligne de code suivante entre les accolades : cookies = cookies + 1;

Ce qui nous donne maintenant ceci : 

function gagnerCookie() {
	cookies += 1;
};

La ligne de code cookies = cookies + 1; demande a l'ordinateur de : prendre la variable cookies, puis de lui dire d'ajouter + 1 au nombre de cookies, rien de plus simple.
