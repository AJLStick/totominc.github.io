Cours 1 : les bases

Mise en place de la page HTML :

Avant de commencer à apprendre et développer notre propre incremental-game, il faut s'assurer d'avoir les bons outils à disposition. On va tenter de créer un petit clone de Cookie Clicker, un clone très simplistique. Nous avons besoin de plusieurs outils :
- D'un éditeur de texte (Sublime Text 1-2-3, Notepad++ ou même bloc-note).
- Utiliser un navigateur internet récent comme Google Chrome, Firefox, Opera ou Safari.
- Un dossier ou stocker tous nos fichiers.
Il est recommandé de regarder dans le code-source des autres pages web afin de mieux apprendre, par exemple, tu peux regarder dans le code-source du tutoriel en ouvrant la console (F12 ou Ctrl+Shift+I) puis dans l'onglet "Elements", ou alors Ctrl+U pour afficher le code-source dans un autre onglet.

A partir d'ici, on peut remarquer quelque chose : une page web est toujours construite avec de l'HTML, du CSS et du Javascript. On va donc commencer a créer nos 3 fichiers : index.html - style.css - main.js
On place ces trois fichiers dans le même dossier, mais il faut maintenant les inclure dans notre page HTML. On fait alors comme ceci pour le CSS : 

<link rel="stylesheet" type="text/css" href="style.css">
Pour le Javascript : 
<script type="text/javascript" src="main.js">

Nos pages CSS et JS (Javascript) sont maintenant liées à notre HTML, nous pouvons commencer à programmer !

Les bases du clic :

Ce que tu apprendras ici agira comme base de tout ce que tu apprendras plus tard dans ce tutoriel, alors assure-toi bien de lire et suivre attentivement. Ce que nous voulons, c'est cliquer sur un bouton et voir le nombre à l'écran augmenter. Ceci nous demande alors plusieurs choses :
- Un élément de l'interface à cliquer (comme un bouton)
- Un peu de "game logic" pour garder un suivi des nombres
- Une fonction en JS nous permettant de faire augmenter (ou réduire) notre chiffre
- Un élément de l'interface qui affichera notre nombre
- Un moyen de mettre à jour cet élément de l'interface

On commence par créer le bouton en lui ajoutant un attribut onclick comme ça : 

<button onclick="cookieClick()">+1 cookie</button>
cookieClick() est le nom de notre fonction, a chaque fois qu'on clique sur le bouton, nous appellons notre fonction cookieClick() et elle est donc éxecutée.

Maintenant il faut garder le suivi de notre chiffre, pour cela on va créer une variable puis la définir à 0, dans notre fichier main.js :

var cookies = 0;
Notre variable est créée, mais notre fonction qui doit faire augmenter notre variable cookies n'est toujours pas créée, on va maintenant s'en occuper. A chaque fois qu'une fonction est appellée en JS, les lignes de code s'y trouvant à l'interieur sont éxecutées. Dans notre cas, nous voulons une fonction qui, lorsqu'elle est appellée, augmente notre variable cookies par 1. Notre fonction cookieClick() doit ressembler à ça :
function cookieClick() {
				    cookies += 1;
				};

L'affichage de notre variable cookies :

Nous voulons maintenant afficher notre nombre de cookies, qui est inscrit dans la variable cookies. On va tout simplement créer un tag span puis lui ajouter un attribut d'ID appellé "affichage-cookies". On doit avoir ceci : 

<span id="affichage-cookies"></span>
Ensuite on va créer une autre fonction appellée updateGame(), cette fonction va nous permettre de rafraichir les éléments d'affichage, par exemple nos variables qu'on souhaite afficher. Dans cette fonction, on va ajouter cette ligne de code : 
document.getElementById("affichage-cookie").innerHTML = cookies;
Cette ligne de code va chercher dans le document HTML pour l'ID spécifié (donc affichage-cookies pour nous), et modifie le contenu du tag. Dans notre cas, nous voulons afficher la variable cookies, donc la ligne de code nous affichera le nombre de cookies. J'ai ajouté après la variable cookies la chaine de caractères " cookies" car ceci nous permettre de voir le mot cookies après le nombre de cookies, et pas seulement voir un nombre. Notre fonction updateStats() devrait ressembler à ça : 
function updateStats() {
				    document.getElementById("affichage-cookie").innerHTML = cookies + " cookies";
				};
Il faut maintenant placer updateStats() dans une loop. Une loop est un chose qui se répète toutes les X secondes. On fait comme ceci pour en créer une : 
window.setInterval(function() {
				    updateStats();
				}, 10);
J'ai inclus le nom de la fonction au milieu car nous voulons que la fonction soit appellée a chaque fois que la loop s'éxecute. Le temps d'une loop se définit en millisecondes, on le met toujours à la fin avant de refermer par des parenthèses notre loop. Notre loop s'éxécute toutes les 10 millisecondes, soit toutes les 0.01 secondes afin d'éxecuter les fonctions (et donc lignes de codes) qui s'y trouvent à l'intérieur.
