Javascript

Introduction :

On peut introduire le Javascript de différentes manières :
- Soit en incluant le Javascript dans un fichier différent et on inclut le fichier à l'HTML.
- Soit en incluant le Javascript directement dans l'HTML.

Par exemple, j'ai crée mon fichier contenant le Javascript, je l'ai appellé "main.js" et placé exactement dans le même répertoire que mon fichier HTML. Pour l'inclure dans l'HTML, j'ajoute le tag : <script src="main.js"><script>

Si je veux inclure directement le Javascript dans le code du fichier HTML, je le place entre les deux tags <script></script>.
Exemple : <script>var x = 0;</script>

Il est recommandé de placer les tags de script à la fin du <body></body>
